## Style Modes

| Mode | Korean | Description |
|---|---|---|
| literal | 직역 | Closest 1:1 translation |
| natural | 자연스러운 번역 | Fluent, idiomatic |
| creative | 의역 | Adapts meaning and cultural refs |

## Rules

1. Technical terms in the table above MUST be translated consistently
2. Code identifiers (function names, variables, file paths) are NEVER translated
3. Acronyms (PDCA, MCP, CLI, API) stay in English
4. When in doubt, prefer the Korean term if one exists in this glossary
5. New terms should be added to this glossary before use in translations
