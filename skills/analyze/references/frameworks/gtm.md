# Go-To-Market Framework

## When to Use
- When designing a go-to-market plan for a new product or feature
- When connecting segments, channels, and messaging into one launch plan

## When NOT to Use
- When the value proposition itself is still unclear
- When the conversation is only about market direction with no channel plan

## Required Sections
1. Target segment
2. Core message
3. Channel strategy
4. Launch motion
5. Success metrics

## Evidence Expectations
- Support segment and channel choices with data or observation
- Include measurable metrics such as CAC, conversion, or adoption

## Recommended Actions
- Recommend 3 execution priorities split across pre-launch and post-launch
