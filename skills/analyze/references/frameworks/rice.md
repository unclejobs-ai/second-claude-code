# RICE Prioritization Framework

## When to Use
- When prioritizing multiple initiatives against one another
- When a decision needs a simple score grounded in reach, impact, confidence, and effort

## When NOT to Use
- When there are too few candidates to compare
- When inputs are so speculative that scoring would be fake precision

## Required Sections
1. Reach: number of users affected per quarter
2. Impact: scale 3=Massive, 2=High, 1=Medium, 0.5=Low, 0.25=Minimal
3. Confidence: percentage (100%=High, 80%=Medium, 50%=Low)
4. Effort: person-months
5. Final RICE Score: Formula = (Reach × Impact × Confidence) / Effort

## Evidence Expectations
- Use one scoring rubric across every item
- Flag low-confidence items that need research before prioritization

## Recommended Actions
- Recommend the top 3 items and the ones that should wait
