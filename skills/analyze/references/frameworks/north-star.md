# North Star Metric Framework

## When to Use
- When defining the core value-creation metric for the organization
- When aligning multiple team metrics to one top-level outcome

## When NOT to Use
- When the goal is short-term reporting rather than value delivery
- When the organization is tracking vanity metrics only

## Required Sections
1. User value event
2. North star metric
3. Input metrics
4. Guardrail metrics
5. Risks / gaming behaviors

## Evidence Expectations
- Explain why the metric represents user value
- Do not propose a single number without guardrails

## Recommended Actions
- Recommend 3 operating actions around the North Star and its inputs
