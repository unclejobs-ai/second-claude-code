# Write Gotchas

## Claude가 콘텐츠 작성에서 자주 실패하는 패턴

### 1. 제네릭 보이스
**증상**: 모든 글이 같은 톤 — "오늘은 ~에 대해 알아보겠습니다"
**해결**: 보이스 가이드(references/voice-guides/)를 반드시 로드 후 작성

### 2. 리서치 건너뛰기
**증상**: 기존 학습 데이터만으로 작성, 최신 데이터 누락
**해결**: /scc:research 호출 필수. Research Brief 없이 작성 시작 금지

### 3. 분량 부족
**증상**: 뉴스레터 800자, 아티클 1500자 — 핵심 건너뛰기
**해결**: 포맷별 최소 분량 강제 (newsletter 2000, article 3000, report 4000)

### 4. CTA 누락
**증상**: 결론이 "이상으로 마치겠습니다"
**해결**: 모든 포맷에 구체적 다음 행동(CTA) 필수

### 5. 패딩
**증상**: 같은 내용을 다른 표현으로 반복하여 분량 채우기
**해결**: 모든 문단이 새로운 정보/인사이트를 추가해야 함. 반복 = 삭제
