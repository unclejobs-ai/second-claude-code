# Shorts/Reels Script Format Reference

## Structure
1. **Hook** (0-3초): 질문 또는 충격적 사실. "알고 계셨나요?" 금지.
2. **Problem** (3-15초): 공감할 수 있는 문제/상황 묘사
3. **Solution** (15-45초): 핵심 인사이트 또는 방법 (3단계 이하)
4. **CTA** (45-60초): 구체적 행동 유도 ("댓글에 ~ 남겨주세요")

## 분량
- 총 300-500자 (60초 기준)
- Hook: 1-2문장
- 각 단계 명확한 전환 표시

## 필수 요소
- 말하기용 구어체 (읽는 글이 아닌 말하는 대본)
- 자막 텍스트 별도 표기
- 비주얼 지시 포함 [화면: ...]
- "여기서 중요한 건요" 같은 전환어
