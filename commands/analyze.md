---
description: "Apply strategic framework (SWOT, RICE, OKR, Lean Canvas...)"
---

Invoke the `scc:analyze` skill to apply a strategic framework.

## Arguments
- First argument: framework name (swot|rice|okr|prd|lean-canvas|persona|journey-map|pricing|gtm|north-star|porter|pestle|ansoff|battlecard|value-prop)
- Second argument: subject to analyze

## Task
Use the Skill tool to invoke `scc:analyze` with the provided framework and subject.
